# How to publish SDK

Run GitHub Actions `npm-publish` to publish

## Run `npm-publish`

1. Select `npm-publish` in [Github Actions](https://github.com/KyuzanInc/mint-sdk-js/actions)
2. `run workflow` with `main` branch
