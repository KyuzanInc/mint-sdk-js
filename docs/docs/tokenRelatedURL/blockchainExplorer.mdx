---
title: BlockchainExplorerのURL生成方法
---

EtherscanなどのURLを提示して、トランザクション詳細をユーザーに示すことができます。

`Txハッシュ`を含む、アイテム関連のトランザクションは、`SDK.getItemLogs()`で取得することができます。

## Etherscan

`NetworkId`が`1, 4`のアイテムのトランザクショントラッキングURLは以下の方法で生成できます

### トランザクションをトラッキングするURLを生成する方法

Mainnet(NetworkId=1)

```typescript
https://etherscan.io/tx/[Txハッシュ]
```

Rinkeby(NetworkId=4)

```typescript
https://rinkeby.etherscan.io/tx/[Txハッシュ]
```

## Matic Explorer

`NetworkId`が`137, 8001`のアイテムのトランザクショントラッキングURLは以下の方法で生成できます

### トランザクションをトラッキングするURLを生成する方法

Mainnet(NetworkId=137)

```typescript
https://explorer-mainnet.maticvigil.com/tx/[Txハッシュ]
```

Mumbai(NetworkId=80001)

```typescript
https://explorer-mumbai.maticvigil.com/tx/[Txハッシュ]
```
