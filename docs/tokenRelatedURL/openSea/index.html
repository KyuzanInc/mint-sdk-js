OpenSeaのNFTページURL生成方法

発行したNFTのOpenSeaページを提示して、ユーザーにNFTの２次流通をしやすくさせたり、NFTの発行証明を提示することができます。

  • コントラクトアドレスは、Item.mintContractAddressです
  • tokenIdは、Item.tokenIdです

tradeType=fixedPriceまたはtradeType=auctionのItemは、買われるか引出されるまでNFTは発行されません。 Item.buyerAddressの値を見ることで、Itemが"買われた"または"引出された"を判断することができます。

OpenSeaのNFTページURL生成方法#

Mainnet(NetworkId=1)#

https://opensea.io/assets/[コントラクトアドレス]/[tokenId]

Rinkeby(NetworkId=4)#

https://testnets.opensea.io/assets/[コントラクトアドレス]/[tokenId]
